
# LuxNLP Raw Dataset

This folder contains the **raw data sources** used in the LuxNLP project before any preprocessing or cleaning.

## 📂 Files Overview

* **Lod_lu.json**
  Raw dataset extracted from LOD.lu (Luxembourgish dictionary).

* **before_wikidata_lb.json**
  Initial Wikidata extraction before cleaning and filtering.

* **wikidata_lb_other_ner1.json**
  Additional Wikidata-based entity data.

## 🧠 Description

These datasets are:

* Unprocessed and not cleaned
* Not tokenized
* Not in CoNLL format

They serve as the **initial input sources** for building the LuxNLP dataset.

## 🎯 Purpose

* Source data for extraction and processing
* Used to generate structured datasets in later stages
* Maintains reproducibility of the pipeline

---

*Note: Data in this folder may contain noise, duplicates, and unstructured content.*
