# 🛠️ Utilities Module

The `utilities/` directory contains helper scripts for converting and transforming data formats.

---

## 🎯 Purpose

* Support data transformation between formats
* Enable easier processing and debugging

---

## 📂 Files Overview

### `conll_to_sentences.py`

Converts CoNLL format into plain text sentences.

---

### `csv_to_json.py`

Converts CSV files into JSON format.

---

## 🔄 Role in Pipeline

```text
Intermediate Data → Utilities → Format Conversion
```

---

## 🧠 Summary

✔ Lightweight helper tools
✔ Format conversion support
✔ Improves pipeline flexibility

---

This module simplifies **data handling and format interoperability**.
