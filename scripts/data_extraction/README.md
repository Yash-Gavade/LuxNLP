# 🌐 Data Extraction Module

The `data_extraction/` directory contains all scripts responsible for **collecting, processing, and transforming raw data** into structured formats for the LuxNLP pipeline.

This module is the **first stage** of the pipeline and forms the foundation for building the Luxembourgish Named Entity Recognition (NER) dataset.

---

## 🎯 Purpose

The data extraction module is designed to:

* Collect entity data from multiple sources (LOD.lu, Wikidata, Leipzig, etc.)
* Extract structured information such as entities, categories, and translations
* Convert raw data into **CoNLL format (BIO tagging)**
* Clean and preprocess extracted text

👉 It ensures high-quality, structured input for dataset creation.

---

## 🧩 Submodules

### 📄 conll_generation

Scripts for converting extracted data into **CoNLL format**.

**Includes:**

* Gazetteer-based CoNLL generation
* Conversion from Leipzig corpus
* Conversion from LOD lexicons
* Conversion from Wikidata

👉 Produces structured NER-ready data.

---

### 📚 lod

Scripts for extracting data from **LOD.lu (Luxembourg Open Data)**.

**Includes:**

* Entity categories extraction
* Translation extraction
* Lexicon creation
* Dataset inspection

👉 Provides high-quality lexical resources for entities.

---

### 🧹 preprocessing

Scripts for cleaning and preparing raw text.

**Includes:**

* Cleaning extracted sentences
* Converting sentences into CoNLL format
* Merging and normalizing data

👉 Ensures data consistency and quality.

---

### 🛠️ utilities

Helper scripts for format conversion and processing.

**Includes:**

* Converting CoNLL to plain sentences
* Converting CSV files to JSON

👉 Supports data transformation across formats.

---

### 🌍 wikidata

Scripts for scraping and extracting entity data from **Wikidata**.

**Includes:**

* Extraction of all entity types
* Extraction of specific categories (e.g., PER)
* Query-based data collection

👉 Provides large-scale entity coverage.

---

## 🔄 Role in Pipeline

```text
Raw Data Sources → Data Extraction → Dataset Creation → Model Training
```

This module transforms unstructured external data into **structured, NER-ready datasets**.

---

## ⚙️ Example Usage

```bash
# Extract data from Wikidata
python scripts/data_extraction/wikidata/wikidata_scrape_all.py

# Generate CoNLL format
python scripts/data_extraction/conll_generation/conll_from_lod.py

# Clean extracted sentences
python scripts/data_extraction/preprocessing/clean_extracted_sentences.py
```
